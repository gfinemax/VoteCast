# RHA VoteCast - 사용자 매뉴얼

지역주택조합 총회 투표 관리 시스템(RHA VoteCast)의 사용 방법을 안내합니다.

## 1. 시작하기
시스템은 웹 브라우저를 통해 접속합니다.
- 권장 브라우저: Chrome, Edge (최신 버전)
- 접속 주소: (배포된 URL 또는 로컬 주소 `http://localhost:3000`)

## 2. 화면 구성
시스템은 크게 두 가지 탭으로 구성됩니다.
1.  **관리자 화면 (Admin):** 총괄 제어 (안건 선택, 화면 전환, 결과 입력)
2.  **입구 확인 화면 (Check-in):** 조합원 입장 처리 전용 **[NEW]**
3.  **송출 화면 (Projector):** 대형 스크린 표시용

시스템 접속 시 나타나는 **'역할 선택 화면'**에서 담당 업무에 맞는 버튼을 클릭하여 입장하십시오.

---

## 3. 입구 안내 요원 가이드 [NEW]
*태블릿 또는 노트북 사용을 권장합니다.*

1.  **조합원 검색:** 화면 상단 입력창에 동/호수 숫자(예: 101-1204)를 입력합니다.
2.  **입장 처리:** 검색된 명단 우측의 파란색 **[입장 확인]** 버튼을 누릅니다.
    - 버튼이 회색 **[입장 완료]**로 바뀌면 처리가 끝난 것입니다.
3.  **실시간 반영:** 처리된 내역은 중앙 관리자 화면으로 즉시 전송됩니다. 별도 보고할 필요가 없습니다.

---

## 4. 관리자 화면 사용법

### 단계 1: 안건 및 화면 모드 선택
- 좌측 사이드바에서 안건을 선택합니다.
- 상단 컨트롤 바에서 **[PPT 보기]** 혹은 **[결과 보기]** 버튼을 눌러 스크린 화면을 제어합니다.

#### 💡 총회 및 안건 관리하기 (고급)
1.  **새 총회 만들기:** 목록 상단의 **[+ 총회 추가]** 버튼을 눌러 새로운 그룹(예: 2027 정기총회)을 만듭니다. 가장 최신 총회가 목록 맨 위에 표시됩니다.
2.  **안건 추가하기:** 각 총회 폴더를 펼치면 하단에 **[+ 안건 추가]** 버튼이 있습니다. 이 버튼을 사용하여 해당 총회에 안건을 추가할 수 있습니다.
3.  **수정 및 삭제:** 총회나 안건 제목 옆의 **✏️(수정)** 아이콘을 눌러 이름을 변경하거나 **🗑️(삭제)** 아이콘으로 제거할 수 있습니다.
4.  **폴더 접기/펼치기:** 총회 제목을 클릭하면 하위 안건 목록을 접거나 펼칠 수 있어 화면을 깔끔하게 유지할 수 있습니다.

### 단계 2: 성원(참석자) 확인
**'02. 성원(참석) 입력'** 섹션을 확인합니다.
- **현장참석:** 입구에서 체크인된 인원이 **실시간으로 자동 카운트**되어 표시됩니다. (수동 수정 불가능)
- **서면결의:** 사전에 집계된 숫자를 수동으로 입력합니다.

### 단계 3: 투표 결과 입력
개표가 완료되면 **'03. 투표 결과 입력'** 섹션에 득표수를 입력합니다.
- **찬성 / 반대 / 기권(무효):** 집계된 표 수를 각각 입력합니다.

> **⚠️ 주의:** 입력된 표의 총합이 '총 참석자' 수와 다를 경우, 붉은색 경고 메시지가 표시됩니다. 오차가 없는지 반드시 확인하세요.

### 단계 4: 결과 송출
- 투표 집계 수치를 입력하고 검증이 통과되면 **[결과 화면 송출하기]** 버튼을 누릅니다.
- 송출 화면이 자동으로 **[결과 보기]** 모드로 전환되며 애니메이션이 재생됩니다.

---

## 5. 송출 화면 활용 팁
- 실제 총회장에서는 **듀얼 모니터**를 사용하는 것을 권장합니다.
- 브라우저 창을 두 개 띄워, 하나는 **관리자 화면(노트북 화면)**, 다른 하나는 **송출 화면(빔프로젝터)** 에 배치하여 운영하면 매끄러운 진행이 가능합니다.
- 송출 화면에서 **F11** 키를 눌러 전체 화면(Full Screen) 모드로 전환하면 더욱 전문적인 연출이 가능합니다.
